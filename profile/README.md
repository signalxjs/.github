SignalX is a lightweight reactive framework family. A small **signals** core powers
components, routing, state, SSR, native UI on Lynx, and even TUIs in the
terminal — all with TSX and full TypeScript inference.

## ⚡ Quickstart

```bash
npm create sigx@latest
```

That's it — pick a template, install, and start hacking.
Full guide → **[signalxjs.github.io/docs](https://signalxjs.github.io/docs)**

| --- | --- |
| [**core**](https://github.com/signalxjs/core) | Reactive component framework — signals, reactivity, TSX |
